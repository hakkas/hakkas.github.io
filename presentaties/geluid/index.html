Hoe groot wordt een geluidsbestand van 3 minuten met een sample rate van 44.1 kHz en een bit depth van 16 bits?

Hoeveel samples in totaal? 44100 * 180 seconden = 7938000 samples

Samples * aantal bits per sample = 7938000 * 16 bits = 127008000 bits

Omzetten naar bytes = 127008000 / 8 = 15876000

Omzetten naar kilobytes = 15876000 / 1024 = 15503 megabytes

Omzetten naar megabytes = 15503 / 1024 = 15 megabytes
